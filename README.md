how to write hello world  in C
```C
printf("Hello world");
```
and
how to write hello world in C++
```Cpp
cout<<"Hello world";
// and 
std::cout<<"Hello world";
```

> C "Hello, World!" program
```C
#include <stdio.h>
int main() {
   printf("Hello, World!");
   return 0;
}

```
> C++ "Hello, World!" program
```C++
#include <iostream>

int main() {
    std::cout << "Hello World!";
    return 0;
}
```
